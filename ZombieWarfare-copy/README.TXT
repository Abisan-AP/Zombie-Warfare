PROJECT TITLE: ZombieWarfare
PURPOSE OF PROJECT: To make a fun Tower Defense game while making the game moderately to difficult in difficulty.
VERSION or DATE: Final Version (Gold Master) 12/10/2020
HOW TO START THIS PROJECT: Open Greenfoot and run the game. Press any buttons available in the screen.
AUTHORS: Gevorg Markarov, Marc Neefa, Abisan Poothapillai, and Roberto Natale
USER INSTRUCTIONS:

Use W,A,S,D to move up, left, down, right respectively.
Press mouse button 1 to shoot the bullet.

You win Level 1 by surviving the 20 Zombie Wave.
You win Level 2 by surviving the 40 Zombie Wave.
You win Level 3 by surviving the 60 Zombie Wave.

Shoot the Zombies before 3 of them reach the Car,
you lose otherwise.
